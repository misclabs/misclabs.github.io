# CSS Notebook ## Box Model Everything is a box. A box is sized *intrinsicly* (implicitly according to content) or *extrinsicly* (explicit). The four areas of the box model are content box, padding box, border box, margin box. Scrollbars will occupy the padding space. The border edge is the limit of what you can see in a box. `outline` and `box-shadow` properties occupy the margin box. For values that have a top,bottom,left,right values: - 0: all four are set - 0 1: 0 is top/bottom, 1 is left/right - 0 1 2: 0 top, 1 is left/right, 2 is bottom - 0 1 2 3: 0 top, 1 right, 2 bottom, 3 left ## References [Learn CSS! on web.dev](https://web.dev/learn/css) [Selectors Guide on MDN](https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Selectors) ## Tags For external use `link` tag in `head`. e.g. `` For embedded use `style` tag in `head`. e.g. `` `style` attribute? ## Rules `RULESET = SELECTORS { DECLARATIONS }` `SELECTOR = tag|.class|#id` `DECLARATION = PROPERTY: VALUE;` ### Other SELECTORS Attribute: `img[src]` selects `` elements, but not `` Pseudo-class: `a:hover` e.g. ``` p, li, h1 { color: red; border: 1px solid black; } ```
